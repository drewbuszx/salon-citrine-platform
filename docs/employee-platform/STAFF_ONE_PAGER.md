## Install

**URL:** https://team.saloncitrineindy.com/team/install

Open that page (or scan the QR on it), then pin Team to your home screen.

| Phone | Browser | Steps |
|-------|---------|--------|
| **iPhone** | **Safari only** | Share → **Add to Home Screen** → crystal icon |
| **Android** | **Chrome** | Menu → **Install app** / **Add to Home screen** |

Chrome on iPhone is **not** a full install path — use Safari.

After install, the crystal icon opens the sign-in screen in its own shell (not a browser tab).

**Sign in:** https://team.saloncitrineindy.com/team/login
